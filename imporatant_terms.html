Here are some important terms related to web development:

  • HTML (Hypertext Markup Language): The standard markup language used to create web pages. It structures the content on the web.
  • CSS (Cascading Style Sheets): A stylesheet language used to describe the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall appearance of web pages.
  • JavaScript: A programming language that enables interactive web pages. It allows developers to create dynamic content, handle user interactions, and manipulate the DOM (Document Object Model).

    web development has 2 category

  • Frontend: The part of a website or web application that users interact with directly. It includes the design, layout, and user interface elements.
  • Backend: The server-side of a web application that handles data processing, database interactions, and business logic. It is responsible for managing the application's functionality.

  • web development used in 2 type:

  • Static Websites(website): Websites that consist of fixed content and are delivered to the user exactly as stored. They are typically built using HTML and CSS.
  • Dynamic Websites(web application): Websites that generate content on-the-fly based on user interactions or server-side processing. They often use server-side scripting languages like PHP, Python, or Ruby.

    • More Important Terms:

  • API (Application Programming Interface): A set of rules and protocols that allows different software applications to communicate with each other. APIs are commonly used to connect frontend and backend systems.
  • Responsive Design: An approach to web design that ensures web pages adapt and display correctly on various devices and screen sizes, including desktops, tablets, and smartphones.
  • CMS (Content Management System): A software application that allows users to create, manage, and modify digital content on a website without requiring specialized technical knowledge. Examples include WordPress, Joomla, and Drupal.
  • Version Control: A system that tracks changes to files and allows multiple developers to collaborate on a project. Git is a popular version control system used in web development.
  • Deployment: The process of making a web application or website available for users on the internet. This typically involves uploading files to a web server.

Internet:A global network of interconnected computers and servers that allows for the exchange of information and communication worldwide. It is the backbone of the World Wide Web (WWW) and enables access to websites, email, social media, and various online services.
  • WWW (World Wide Web): A system of interlinked hypertext documents and resources accessed via the internet. It allows users to view and navigate web pages using web browsers.
  • URL (Uniform Resource Locator): The address used to access a specific resource on the internet, such as a web page. It typically consists of the protocol (e.g., http or https), domain name, and path to the resource.
  • HTTP (Hypertext Transfer Protocol): The protocol used for transmitting hypertext (web pages) over the internet. It defines how messages are formatted and transmitted between web servers and clients (browsers).
  • HTTPS (Hypertext Transfer Protocol Secure): An extension of HTTP that adds a layer of security by encrypting data exchanged between the web server and client using SSL/TLS protocols. It ensures secure communication and protects sensitive information.
  • IP Address (Internet Protocol Address): A unique numerical identifier assigned to each device connected to a computer network that uses the Internet Protocol for communication. It allows devices to locate and communicate with each other on the internet.
  • DNS (Domain Name System): A hierarchical system that translates human-readable domain names (e.g., www.example.com) into IP addresses. It acts as a directory for the internet, allowing users to access websites using easy-to-remember names instead of numerical IP addresses.
  • Web Server: A computer or software application that hosts websites and delivers web pages to users' browsers upon request. It processes incoming requests, retrieves the requested resources, and sends them back to the client.
  • Web Browser: A software application used to access and view web pages on the internet. Examples include Google Chrome, Mozilla Firefox, Safari, and Microsoft Edge.
