HTML & CSS Important Terms

A beginner-friendly guide for Web Development

HTML Important Terms

  • HTML: HyperText Markup Language used to create web page structure.
  • Tag: Keywords enclosed in angle brackets < >.
  • Element: Combination of opening tag, content, and closing tag.
  • Attribute: Provides extra information about an element.
  • DOCTYPE: Defines the HTML version.
  • Head: Contains metadata, title, and CSS links.
  • Body: Contains visible content of the webpage.
  • Block Elements: Elements that take full width.
  • Inline Elements: Elements that take required space only.
  • Semantic Tags: Meaningful tags like header, footer, section.
  • Form: Used to collect user input.
  • Input: Used to take data from user.
  • Anchor Tag: Used to create hyperlinks.
  • List: Ordered and unordered list.
  • Table: Used to display data in rows and columns.
  • ID: Unique identifier for an element.
  • Class: Used to style multiple elements.

CSS Important Terms

  • CSS: Cascading Style Sheets used for styling HTML.
  • Selector: Selects the HTML element to style.
  • Property: Style attribute like color, width.
  • Value: Assigned value to a property.
  • Declaration: Property and value pair.
  • Rule Set: Selector with declarations.
  • Inline CSS: CSS written inside HTML element.
  • Internal CSS: CSS written inside <style> tag.
  • External CSS: CSS written in a separate file.
  • Box Model: Content, padding, border, margin.
  • Margin: Space outside the element.
  • Padding: Space inside the element.
  • Border: Outline around the element.
  • Display: Defines element display type.
  • Position: Defines element position.
  • Flexbox: One-dimensional layout system.
  • Grid: Two-dimensional layout system.
  • Pseudo-class: Defines special state like hover.
  • Pseudo-element: Styles specific part of element.
  • Media Query: Used for responsive design.
  • Z-index: Controls element stacking order.
  • Overflow: Controls content overflow.
  • Opacity: Controls transparency.
  • Transition: Smooth change effect.
